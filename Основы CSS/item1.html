Деактиватор (Jammer)

Краткое описание товара

Устройство, создающее направленное поле помех. Необходим для прохода через заблокированные зоны.

Характеристики

  • Тип: Генератор помех
  • Радиус действия: Ограничен прямой видимостью
  • Питание: Автономное
  • Статус: Разблокировано

Подробное описание товара

Деактиватор используется для отключения силовых полей, автоматических турелей и боевых дронов. Позволяет решать сложные пространственные головоломки Архива, временно нейтрализуя угрозы симуляции.
