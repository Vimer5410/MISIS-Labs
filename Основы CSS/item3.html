Гексаэдр (Hexahedron)

Краткое описание товара

Многофункциональный кубический блок, используемый для взаимодействия с физикой среды.

Характеристики

  • Материал: Металлический сплав
  • Вес: Стандартный для нажимных механизмов
  • Тип: Геометрический примитив

Подробное описание товара

Используется в качестве надежного противовеса для нажимных плит, удерживая их в активном состоянии. Также может служить подставкой для достижения высоких точек симуляции, блокировки лазерных лучей или остановки мобильных дронов.
