Деактиватор (Jammer)

Краткое описание модуля

Устройство, создающее направленное поле помех. Необходим для прохода через заблокированные зоны.

Технические данные

  • Тип: Генератор помех
  • Радиус действия: Ограничен прямой видимостью
  • Питание: Автономное
  • Статус: Разблокировано

Подробная информация

Данное устройство применяется для временного отключения электронных преград: силовых полей, пулеметов и активных мин. При перемещении деактиватора его действие прекращается.
