Гексаэдр (Hexahedron)

Краткое описание модуля

Многофункциональный кубический блок, используемый для взаимодействия с физикой среды.

Технические данные

  • Материал: Металлический сплав
  • Вес: Стандартный для нажимных механизмов
  • Тип: Геометрический примитив

Подробная информация

Используется для активации нажимных плит, блокировки лучей или как подставка для преодоления невысоких преград. Может быть установлен на другие устройства или перемещен потоком воздуха.
